# 🧟 Dead Zone

A 2D zombie survival shooter built with Python and Pygame.

## Features

- Adaptive Zombie AI
- Three Unique Zombie Types
- Weapon & Reload Mechanics
- Procedurally Generated Sound Effects
- Blood Particle Effects
- Large Scrollable World
- Health, Ammo & Score System
- Mouse-Based Aiming
- Dynamic Enemy Behavior

## Controls

- WASD = Move
- Mouse = Aim
- Left Click = Shoot
- R = Reload
- ESC = Pause
